# AI Performance Baseline Validator

## Project Overview

AI Performance Baseline Validator is an AI-driven platform monitoring solution developed as part of pSiddhi Semester 2 (Platform Track).

The project monitors microservice performance metrics, generates statistical baselines using moving average and z-score techniques, detects anomalies, and provides AI-generated explanations through dashboards.

## Technology Stack

- Python
- FastAPI
- Prometheus
- Grafana
- MLflow
- Azure Functions
- Gemini API
- Power BI
- GitHub Actions
- Pytest
